# GTNH Trans Pride Resourcepack

A resourcepack for GregTech New Horizons that retextures various commonly-used tools to be white, cyan, and pink

- Trans Tools: The base resource pack
- Trans Tools Static: Overrides the base resource pack's animated textures with static variants
- Alt Trans Vajra: A Vajra based off of the old texture
